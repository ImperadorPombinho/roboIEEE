ROBÔ BOISU

o que é um robô?

robô é uma maquina capaz de fazer tarefas, utilizada muitas vezes para ajudar humanos em tarefas repetitivas e mecanicas.

Quem é Boisu?

Boisu é um robô, capaz de citar frases em libras, como "Oi eu sou Boisu"

Sua missão é "trazer para as pessoas uma introdução sobre libras"

Assim fazendo os sinais de forma dinamica para executar tal missão

Logica de como fazer o robo se movimentar

a logica pensada, foi em base com o braço do corpo humano, onde para se movimentar o braço o cerebro envia sinais nervosos para o local, fazendo este se movimentar para posição desejada, no robo foi pensado da mesma forma, mas convertido em numero, onde cada letra teria dados numericos especificos para cada motor do braço, fazendo um laço de repetição interagir com os motores, enviando cada dado para cada motor imitando o cerebro humano

Beneficios do Boisu na Sociedade

  • Introduzir Libras nas Empresas
  • Auxiliar o ensinamento de Libras
  • Facilidade na manutenção de sua programação
  • Mordenização na educação
